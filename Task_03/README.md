# 🔑 Task 03 – Password Strength Checker

This project is a Password Strength Checker built using Python. It evaluates the strength of a password based on various security criteria.

## 🚀 Features
- Checks password length
- Detects uppercase and lowercase letters
- Verifies presence of numbers and special characters
- Provides strength feedback (Weak / Medium / Strong)

## ⚙️ How to Run
```bash
python3 password_checker.py
