# ⌨️ Task 04 – Educational Keylogger

This project is an Educational Keylogger built using Python. It captures and logs keystrokes to demonstrate how input monitoring works in cybersecurity.

## 🚀 Features
- Captures keystrokes in real-time
- Logs both normal and special keys
- Saves keystrokes to a file
- Optional GUI interface for better usability

## ⚙️ How to Run
```bash
python3 keylogger.py
