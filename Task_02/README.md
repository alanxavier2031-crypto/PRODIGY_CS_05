# 🖼️ Task 02 – Image Encryption Tool

This project is an Image Encryption Tool built using Python. It encrypts and decrypts images by modifying pixel values, ensuring basic image security.

## 🚀 Features
- Encrypt image using pixel manipulation
- Decrypt image back to original form
- Supports common image formats (PNG, JPG)
- Simple and easy-to-use implementation

## ⚙️ How to Run
```bash
python3 image_encryptor.py
